# 📚 Библиотечен мениджър

Python проект за управление на малка библиотека — добавяне, заемане, връщане и търсене на книги.

---

## Структура на проекта

```
library_manager/
├── models.py         # Класовете Book и Library
├── main.py           # Демонстрационен скрипт
├── requirements.txt  # Зависимости (само stdlib)
└── README.md         # Тази документация
```

---

## Изисквания

| Компонент | Версия |
|-----------|--------|
| Python    | ≥ 3.10 |
| Библиотеки | само стандартна (datetime) |

## Описание на класовете

### `Book` (models.py)
Представлява книга с атрибути:

| Атрибут | Тип | Описание |
|---------|-----|----------|
| `title` | str | Заглавие |
| `author` | str | Автор |
| `year` | int | Година на издаване |
| `genre` | str | Жанр |
| `copies` | int | Общ брой копия |
| `available_copies` | int | Налични копия |
| `borrowed_by` | list | Наематели |

### `Library` (models.py)
Управлява колекция от книги.

| Метод | Описание |
|-------|----------|
| `add_book(book)` | Добавя книга; обединява дублирани заглавия |
| `borrow_book(title, borrower, days)` | Регистрира заемане с краен срок |
| `return_book(title, borrower)` | Регистрира връщане |
| `list_books(sort_by, only_available)` | Сортиран списък (title / author / year / genre) |
| `search(query)` | Търсене по заглавие, автор или жанр |
| `statistics()` | Обобщена статистика |
